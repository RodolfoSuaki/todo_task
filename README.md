# Todo App com HTML, CSS e JavaScript

Este projeto implementa um aplicativo simples de lista de tarefas (Todo App) utilizando HTML, CSS (Tailwind CSS) e JavaScript, com armazenamento de dados no `localStorage`. O projeto permite a criação, leitura, atualização e exclusão de tarefas, persistindo as informações no navegador para que não sejam perdidas ao recarregar a página.

## Como o projeto foi desenvolvido

O projeto foi construído com as seguintes tecnologias:

* **HTML:** Define a estrutura da página e os elementos da interface do usuário, como o campo de entrada de texto, o botão de adicionar tarefa e a lista de tarefas.
* **CSS (Tailwind CSS):**  Utilizado para estilizar a aplicação, fornecendo uma aparência moderna e responsiva.  O Tailwind CSS facilita a estilização com classes utilitárias.
* **JavaScript:** Implementa a lógica do aplicativo, incluindo a manipulação do DOM para atualizar a interface, o gerenciamento das tarefas no array `savedTasks`, a interação com o `localStorage` para persistência de dados e o tratamento de eventos.

A estrutura do projeto é simples, com os arquivos `index.html`, `style.css` e `script.js` no mesmo diretório.  O arquivo HTML inclui os links para o CSS do Tailwind e o arquivo CSS personalizado. O JavaScript é carregado no final do body do HTML.
